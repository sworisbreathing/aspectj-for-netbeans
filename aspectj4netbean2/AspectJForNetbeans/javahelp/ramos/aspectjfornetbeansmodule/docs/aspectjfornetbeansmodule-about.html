About The Aspectj For NetBeans Module

This module project is a fork of the AspectJ Development Environment (AJDE) module extension to NetBeans project hosted at SourceForge.net. See also the homepage of the original version: http://aspectj-netbeans.sourceforge.net/. The original version was made by Mik Kersten, who is the project owner. The module depends on a library module that contains aspectj1.5.3

Testing the module

To test this preview module you'll need a project, with one or more ".lst" configuration files that apply to the whole project (i.e to all java files), set as the main project.

You need also the aspectjrt.jar file from the aspectj 1.5.3 distribution set in the class path of the project.

You first have to start AJDE by pressing the toolbar button or the menu item under the tools menu. The structure browser appears.

You then select one of the configuration files from the aspectj menu. Then the aspectj compiler and the IDE will build your project. If there were no errors you can navigate in the structure browser.
